# YouStack Start

Use this starter skill when a local agent begins work with a personal YouStack.

1. Read the local `youstack.json` manifest.
2. Run `youmd stack smoke --path <stack>` before trusting generated files.
3. Load local project instructions and project-context files first.
4. Use shared You.md API/MCP only for protected memory, private context, sync, or connected actions.
5. Do not write to the brain, connected tools, or repo files until the user approves the exact action.
