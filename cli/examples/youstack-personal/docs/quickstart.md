# Personal Agent Start Quickstart

```bash
youmd stack inspect --path cli/examples/youstack-personal
youmd stack smoke --path cli/examples/youstack-personal
youmd stack capabilities --path cli/examples/youstack-personal
youmd stack route --path cli/examples/youstack-personal "start this repo with my preferences"
```

This starter stack is local-only by default. Protected memory and private context stay behind You.md API/MCP until a scoped grant exists.
